# SharePulse

A real-time web application for tracking Nepal Stock Exchange (NEPSE) stocks with personalized watchlists and market insights.

## Features 

- **Watchlist** - Add and track your favorite stocks with real-time price updates
- **Market Movers** - View top gainers, losers, and most traded stocks
- **All Stocks** - Browse complete list of NEPSE stocks with live data
- **Dark Mode** - Toggle between light and dark themes
- **Responsive Design** - Optimized for both mobile and desktop devices

## Tech Stack

- **Next.js 15** - React framework
- **Tailwind CSS** - Styling
- **Heroicons** - UI icons
- **Local Storage** - Persistent watchlist data

## Data Source

Stock data is fetched from the official NEPSE API at regular intervals to provide real-time updates.
